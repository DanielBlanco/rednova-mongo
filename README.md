# ZONGO
ZIO thin wrapper for Mongo4Cats.

## Setup

Just import the library into your Scala dependencies.

```
"dev.dblancorojas" %% "zongo-core"  % "0.1.0"
"dev.dblancorojas" %% "zongo-circe" % "0.1.0"
```
