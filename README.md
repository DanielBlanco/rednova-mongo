# ZONGO
Helper methods to use Mongo4Cats with ZIO.

I'm using this for my own projects waiting for someone to provide a better Library
that I can use.

## Setup

Just import the library into your Scala dependencies.

```
"dev.rednova" %% "mongo" % "0.1.0",
```
